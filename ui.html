Instructions:

This is the initial setup wizard for the Train Approach Bus Indicator Light (TABIL) 2.0. The HTML form on this page will generate a configuration file that will be stored in the 'folder' with the other files used to run the script. To properly set up TABIL 2.0, ensure that the track circuit IDs for your station are in the 'track-circuits.csv' file. Then, fill in the desired values for the following:

  • Minimum Time Between Activations (how long before the light is allowed to turn on after turning off)
  • Arrival Countdown (Countdown timer value in seconds once the train occupies the selected approach circuit)
  • Platform Countdown (Countdown timer value in seconds once the train occupies the selected platform circuit)
  • The weekday and weekend time ranges (times during the day when the TABIL will be active, must be entered in whole numbers between 0 and 23)

Click the "Generate" button to create the setup file.
